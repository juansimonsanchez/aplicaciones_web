WEB
Conjunto de documentos, páginas y recursos interconectados que se acceden a través de Internet mediante un navegador, usando principalmente el protocolo HTTP o HTTPS.
SERVIDOR
Computadora (física o virtual) que almacena, procesa y distribuye información o servicios a otros equipos llamados “clientes”. Ejemplo: un servidor web entrega páginas web a los navegadores.
HOSTING
Servicio que ofrece espacio y recursos en un servidor para alojar sitios web, aplicaciones o archivos, haciéndolos accesibles desde Internet.
INTERNET
Red global de redes interconectadas que permite la comunicación, intercambio de datos e información entre dispositivos en todo el mundo, usando protocolos estandarizados.
